# A/B Test Analysis: Cookie Cats Retention

## Описание проекта
В данном проекте проводится анализ A/B-теста в мобильной игре **Cookie Cats**.  
Команда продукта тестировала изменение в пользовательском пути (перенос игрового "гейта")  
и оценивала влияние новой механики на удержание пользователей.



## Гипотеза
Изменение пользовательского пути влияет на удержание пользователей (retention).



## Метрики

**Основная метрика:**
- `retention_7` — удержание пользователей на 7-й день

**Дополнительные метрики:**
- `retention_1` — удержание пользователей на 1-й день
- `sum_gamerounds` — вовлечённость пользователей



## Данные
- Размер датасета: **90 189 пользователей**
- Группы:
  - `gate_30` — контрольная группа
  - `gate_40` — тестовая группа
- Источник: публичный датасет Cookie Cats

Каждая строка соответствует одному пользователю.



## Этапы анализа

### 1. Проверка данных
- Проверено количество уникальных пользователей
- Пропусков и дубликатов не обнаружено

### 2. Проверка баланса групп
- Размеры контрольной и тестовой групп сопоставимы
- Распределение игровой активности схоже
- Рандомизация признана корректной

### 3. Анализ retention
| Метрика | Control | Test | Абс. разница | Отн. разница |
|------|------|------|------|------|
| retention_1 | 44.82% | 44.23% | -0.59 п.п. | -1.32% |
| retention_7 | 19.02% | 18.20% | -0.82 п.п. | -4.31% |

### 4. Проверка статистической значимости
- Использован χ²-тест
- `retention_7`: **p-value = 0.0016** (статистически значимо)
- `retention_1`: **p-value = 0.0755** (не значимо)

### 5. Доверительные интервалы (Bootstrap)
- 95% CI для разницы retention_7: **[-1.33 п.п. ; -0.32 п.п.]**
- 0 не входит в интервал → эффект устойчив

### 6. Проверка устойчивости
- Удалён 1% пользователей с наибольшим числом игровых раундов
- Результаты и выводы сохранились



## Вывод
Новая механика **негативно влияет на 7-day retention**.  
Снижение статистически значимо и устойчиво.

**Рекомендация:** не внедрять изменение в текущем виде.
